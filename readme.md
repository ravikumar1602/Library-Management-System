# Library Management System

A Library Management System developed using PHP and MySQL.

## Features

- Admin Login & Logout
- Student Management
- Book Management
- Book Issue & Return
- Fine Calculation System
- Dashboard Statistics
- Search Functionality
- Fine Reports

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Modules

1. Authentication
2. Student Management
3. Book Management
4. Issue Book
5. Return Book
6. Fine Report
7. Dashboard

## Installation

1. Clone the repository
2. Import database
3. Configure database.php
4. Run using XAMPP
